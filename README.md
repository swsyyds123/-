# 导航页应用

一个纯前端实现的现代化导航页应用，具备丰富的自定义功能和精美的视觉效果。

## 功能特性

### 前台导航
- 分类展示：将导航链接按分类分组展示
- 搜索功能：支持实时搜索过滤链接
- 快捷访问：点击链接直接跳转

### 后台管理
- 分类管理：添加、编辑、删除导航分类
- 链接管理：添加、编辑、删除导航链接
- 即时预览：修改后立即在前台显示

### 个性化设置
- 主题颜色：6种预设主题色可选
- 背景样式：4种背景样式（3种渐变+1种纯色）
- 字体大小：小、中、大三种尺寸
- 布局样式：网格布局、列表布局

### 数据管理
- 数据持久化：使用 localStorage 自动保存配置
- 导入导出：支持配置数据的导入和导出

## 技术实现

- **框架**: 纯原生 JavaScript，无需依赖第三方框架
- **样式**: CSS3 + Flexbox/Grid 布局
- **图标**: Font Awesome 6.4.0
- **字体**: Google Fonts - Inter
- **存储**: localStorage

## 快速开始

### 方式一：直接打开
1. 下载所有文件到本地目录
2. 用浏览器打开 `index.html` 即可使用

### 方式二：本地服务器
```bash
# 使用 Python 启动简易服务器
python -m http.server 8000

# 或使用 Node.js
npx serve .

# 然后访问 http://localhost:8000
```

## 使用说明

### 前台操作

1. **浏览导航**: 页面展示所有分类，点击链接直接跳转
2. **搜索链接**: 在搜索框输入关键词，实时过滤链接
3. **打开设置**: 点击齿轮图标打开个性化设置面板
4. **进入管理**: 点击"管理"按钮进入后台管理界面

### 设置面板

1. **主题颜色**: 点击颜色圆点选择主题色
2. **背景样式**: 选择渐变或纯色背景
3. **字体大小**: 调整页面文字大小
4. **布局样式**: 在网格和列表布局间切换
5. **保存设置**: 点击保存按钮保存配置
6. **重置默认**: 恢复默认设置

### 后台管理

#### 分类管理
- **添加分类**: 点击"添加分类"按钮，填写名称、图标和排序序号
- **编辑分类**: 点击分类旁的编辑图标进行修改
- **删除分类**: 点击分类旁的删除图标，确认后删除（会同时删除该分类下的所有链接）

#### 链接管理
- **添加链接**: 点击"添加链接"按钮，选择分类并填写链接信息
- **编辑链接**: 点击链接旁的编辑图标进行修改
- **删除链接**: 点击链接旁的删除图标，确认后删除

#### 导入/导出
- **导出配置**: 点击"导出配置"按钮，下载 JSON 格式的配置文件
- **导入配置**: 点击"导入配置"按钮，选择 JSON 文件导入（会覆盖现有数据）

## 图标使用

应用使用 Font Awesome 图标，在添加分类或链接时可自定义图标：

1. 访问 [Font Awesome](https://fontawesome.com/icons)
2. 选择图标，复制图标类名（如 `fas fa-search`）
3. 在添加/编辑表单中粘贴到图标输入框

## 响应式支持

| 设备类型 | 屏幕宽度 | 布局特点 |
|---------|---------|---------|
| 桌面端 | > 768px | 多列网格布局 |
| 平板 | 480-768px | 单列布局 |
| 移动端 | < 480px | 紧凑单列布局 |

## 注意事项

1. 数据存储在浏览器的 localStorage 中，清除浏览器数据会导致配置丢失
2. 建议定期导出配置文件作为备份
3. 导入配置会覆盖现有数据，请谨慎操作
4. 链接地址必须以 `http://` 或 `https://` 开头
